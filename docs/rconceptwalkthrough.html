Filtering and Grouping Tutorial

First we’ll load in the packages needed to analyze the data.

load("ncaa_football_attendance.RData")

Now that we’ve loaded the packages in, we can begin to analyze the data. We’ll start off by filtering the data to only include data for teams in the Southeastern Conference. To do this, we will use the filter function to find “Conference=SEC”

Now that we’ve learned to filter in order to find attendance numbers for all teams in the SEC, lets find what the average attendance was for the SEC in the latest year, which was 2018 (labeled as “year six”). To find numbers for 2018, we’ll use the same “filter” function to focus on just the SEC. After this, we will use the summarize function to find the average attendance across the during 2018.

ncaa_football_attendance %>% 
  filter(Conference == "SEC") %>% 
  summarise(average_SEC_atendance = mean(year_six))
# A tibble: 1 × 1
  average_SEC_atendance
                  <dbl>
1               514639.

Now lets look at the full sheet of data. Lets try to find the mean attendance across the NCAA during the 2018 season.

ncaa_football_attendance %>% 
  summarize(mean_attendance_ncaa = mean(year_six))
# A tibble: 1 × 1
  mean_attendance_ncaa
                 <dbl>
1                   NA

As you can see, this results in a value of “N/A”. This is because several of the dataset are not recorded. Therefore, to find the mean for all of the teams, we’ll have to add “na.rm = TRUE” in order to get rid of the NA values.

ncaa_football_attendance %>% 
  summarize(mean_attendance_ncaa = mean(year_six, na.rm = TRUE))
# A tibble: 1 × 1
  mean_attendance_ncaa
                 <dbl>
1              260592.

Now lets look at conferences again. Lets try to find which conference has the most teams. To do this we’ll utilize the grouping function to find the number of teams in each conference. From there, we can sort to find the conference with the most teams.

ncaa_football_attendance %>% 
  group_by(Conference) %>% 
  summarise(count = n()) %>% 
  arrange(desc(count))

This is a good start; however, when looking at the datasheet we can see that several teams are listed twice. This is because they changed conferences over the period the research was being conducted. To find the teams which changed conferences, we can use the grouping function to find which teams are listed multiple times in the dataset.

ncaa_football_attendance %>% 
  group_by(Institution) %>%

Finally, lets find the average attendance per game for all Division 1 teams during the past year (year_six). Given that there was 815 home games played in Division 1 during the year, we can find the average per game by first calculating the total attendance for the year.

ncaa_football_attendance %>% 
  summarise(total_attendance = sum(year_six, na.rm = TRUE))
# A tibble: 1 × 1
  total_attendance
             <dbl>
1         33876902

Now we can use the total attendance to find the average attendance per game, simply by dividing the total attendance by the total # of games, 815.

ncaa_football_attendance %>% 
  summarise(total_attendance = sum(year_six, na.rm = TRUE)) %>% 
  summarise(total_attendance/815)
# A tibble: 1 × 1
  `total_attendance/815`
                   <dbl>
1                 41567.
